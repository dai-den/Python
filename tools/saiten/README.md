# C言語のプログラムを実行するPythonプログラム
学内向けに作成しています.

## saiten1.py
  個人名(フォルダ)/学籍番号.zip/学籍番号(フォルダ)
  
  のような形式のとき,エラーチェックの上学籍番号(フォルダ)を別のフォルダにコピーするプログラム

## saiten2.py
  学籍番号(フォルダ)/01.c , 02.c , ... 
  
  の形式のときに各プログラムをコンパイルして実行し,結果とエラーを記録するプログラム.

  コンパイルオプションも指定できます.

詳しいフォルダ構成や実行例はexample以下を見てください.
